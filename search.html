Sublime Text

Sublime Text — «лёгкий» редактор исходного кода, разработанный компанией Sublime HQ для Windows, Linux и macOS. Редактор оснащён менеджером пакетов, который позволяет пользователю находить, устанавливать, обновлять и удалять пакеты без перезагрузки программы. Visual Studio

MICROSOFT

Visual Studio — редактор исходного кода с поддержкой технологии IntelliSense и возможностью простейшего рефакторинга кода, разработанный компанией Microsoft для платформ Windows, macOS, Linux. Visual Studio Code

MICROSOFT

Visual Studio Code — редактор исходного кода, разработанный Microsoft для Windows, Linux и macOS. Позиционируется как «лёгкий» редактор кода для кроссплатформенной разработки веб и облачных приложений.
